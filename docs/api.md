# API 接口文档

## 1. Ping 健康检查

### 接口说明
用于检测服务是否正常运行，返回固定字符串 `pong`。

### 请求方式
`GET`

### 请求地址
`/ping`

### 请求参数
无

### 请求头信息
无特殊要求

### 响应格式
`text/plain`

### 响应字段
| 字段 | 类型 | 说明 |
|------|------|------|
| （响应体） | string | 固定返回 `pong` |

### 响应示例
```
pong
```

### 错误示例
服务不可达时返回连接错误（非 HTTP 层错误码），无 JSON 错误体。

---

> 服务监听地址：`:9999`
