# Change: 字符串按空格拆分接口

## 为什么（Why）
接口调用方需要将一个包含多个词的长字符串参数拆分为独立的字符串数组，以便后续逐项处理。目前缺少这一公共工具函数。

## 变更内容（What Changes）
- 新增 `SplitBySpace(s string) []string` 函数，接受一个字符串，按空格拆分后返回字符串数组
- 连续空格视为单个分隔符，首尾空格不产生空元素

## 影响范围（Impact）
- 新增工具函数，无现有逻辑修改
- 影响范围：新增文件 `stringutils/split.go` 及对应测试文件
- 无外部依赖，无 API 破坏性变更
