# 设计：API 文档结构

## 文档位置
`dspec/specs/api/spec.md`

## 文档格式
每个接口按以下结构组织：

```markdown
## 序号. 接口名称

### 接口说明
简要描述接口功能

### 请求方式
`GET/POST`

### 请求地址
`/path`

### 请求参数
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|

### 响应字段
| 字段 | 类型 | 说明 |
|------|------|------|

### 响应示例
```json
{ ... }
```

### 错误示例
```json
{ ... }
```
```

## 接口分类
1. **基础接口**：/、/ping、/test
2. **计算接口**：/add、/variance、/md5
3. **工具接口**：/random、/split、/timestamp、/timezone

## 注意事项
- `/ping` 接口返回值根据时间段变化（10:00-18:00 返回 pong，其他时间返回 pang）
- 所有 POST 接口请求体为 JSON 格式
- 错误响应使用 HTTP 状态码表示
