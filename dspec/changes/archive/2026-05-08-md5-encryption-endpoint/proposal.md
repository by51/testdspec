# Change: 添加 MD5 加密端点

## 为什么（Why）
系统当前缺少字符串加密能力，无法为客户端提供常用的哈希计算服务。MD5 虽然不再适合用于安全场景，但在数据指纹、一致性校验等非安全场景下仍有广泛应用。

## 变更内容（What Changes）
- 新增 `/md5` HTTP 端点，接收字符串输入，返回 MD5 哈希值
- 支持 POST 方法，请求体为 JSON 格式
- 返回 32 位小写十六进制哈希字符串

## 影响范围（Impact）
- 新增端点：`POST /md5`
- 无破坏性变更，不影响现有端点
- 依赖 Go 标准库 `crypto/md5`，无需引入外部依赖
