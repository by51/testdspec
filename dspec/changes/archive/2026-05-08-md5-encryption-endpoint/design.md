# Design: 添加 MD5 加密端点

## 技术方案
在 `main.go` 中新增 `md5Handler` 函数，使用 Go 标准库 `crypto/md5` 进行哈希计算。

方案选择理由：
- 直接在 `main.go` 添加处理器，与现有端点（`/timestamp`、`/timezone`）风格一致
- 使用标准库，无外部依赖，维护成本低
- 返回小写十六进制格式，符合主流 MD5 表示习惯

## 模块结构
```
main.go
├── md5Handler()        // 新增：处理 POST /md5 请求
├── Md5Request          // 新增：请求结构体
└── Md5Response         // 新增：响应结构体
```

## 关键决策
1. **请求方式**：选择 POST 而非 GET，因为输入字符串可能很长，且可能包含特殊字符
2. **响应格式**：返回 JSON 格式，与现有端点风格统一
3. **错误处理**：缺少 input 字段时返回 400 错误，与 `/add`、`/variance` 等端点行为一致
4. **不创建独立包**：MD5 计算逻辑简单，直接在 handler 中实现，避免过度设计
