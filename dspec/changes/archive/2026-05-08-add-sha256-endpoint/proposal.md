# Change: 添加 SHA256 哈希计算端点

## 为什么（Why）
系统目前只提供 MD5 哈希计算能力，MD5 已被证明存在安全漏洞（碰撞攻击），不适合用于安全敏感场景。SHA256 是更安全的哈希算法，广泛应用于密码存储、数据完整性校验等场景。

## 变更内容（What Changes）
- 新增 `POST /sha256` 接口，接收字符串输入，返回 SHA256 哈希值（64 位小写十六进制）
- 请求/响应格式与现有 MD5 接口保持一致

## 影响范围（Impact）
- **新增模块**：无新模块，在 main.go 中新增处理函数
- **API 变更**：新增 `/sha256` 端点
- **依赖变更**：无新增外部依赖（使用 Go 标准库 `crypto/sha256`）
- **风险点**：低风险，新增功能不影响现有功能
